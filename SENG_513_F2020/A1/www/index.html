Preface

Welcome to my blogs, I am Haohu and this blog is a collection of some of my coding experience and skills used in coding competition. Unlike those WIKI of algorithms, this blog just contains some techniques or introductions of some data structures I found interesting and not that hard to understand for a CS student. All the code here are successfully tested in the server of UofC by running Secure Shell.

All the code in all my four articles are written in C++, so why C++? Well, here are some reasons:

  • C++ is faster and easy to tweak than other languages in real coding competitions.

  • C++ has STL, which implements a lot of built-in data structures and algorithms and thus bring much convenience to coders.

  • You can directly use CPU instructions or inline assembly code in C++ for optimization, even though such case is pretty rare.

There are other advantages of using C++ in coding competition, since it can help you get in touch with the hardware more easily than other languages.

On the other hand, the collection of my articles is made of 4 parts:

  • Firstly I introduce how can a coding competitor reduce their run-time in C++ when they have to encounter large size of input/output in competitions.

  • Secondly I introduce the application of Modulo Operation in coding competition.

  • In the third article I share how to use a compiler extension to save coder's time.

  • Finally I give a short introduction on the disjoint set, which is a useful data structure and some interesting stories help you to understand it.
